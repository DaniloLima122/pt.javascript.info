# Obtendo o atributo

Escreva o código para obter do documento o elemento com o atributo `data-widget-name` e leia seu valor.

```html run
<!DOCTYPE html>
<html>
<body>

  <div data-widget-name="menu">Choose the genre</div>

  <script>
    /* seu código */
  </script>
</body>
</html>
```
